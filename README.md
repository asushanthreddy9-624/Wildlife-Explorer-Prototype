# Wildlife Explorer Prototype
**Singapore Wildlife Conservation Mobile App**

A mobile-first prototype for exploring Singapore's wildlife and reporting emergencies.

## 🌐 **Live Demo**
**[🚀 Try the Live App Here](https://asushanthreddy9-624.github.io/Wildlife-Explorer-Prototype/)**

## 🚀 Quick Start

Simply open `mobile-app.html` in your web browser - the app works immediately without any setup!

```bash
# Clone and open
git clone <repository-url>
cd Wildlife-Explorer-Prototype
open mobile-app.html
```

## 📱 Features

### 🌿 Explorer Role (Liz)
- **Live Map**: View wildlife sightings across Singapore
- **Explore**: Discover nature reserves with wildlife information  
- **Identify**: Use AI camera to identify species
- **Report**: Submit emergency wildlife reports

### 🛡️ Admin Role (Max)
- **Dashboard**: View and manage emergency reports
- **Analytics**: Track report statistics and status
- **Management**: Update report status and clear data

## 🦆 Singapore Wildlife Database

### AI Species Identification
- **Smooth-coated Otter** (Lutrogale perspicillata) - 98% confidence
- **Long-tailed Macaque** (Macaca fascicularis) - 95% confidence
- **Oriental Pied Hornbill** (Anthracoceros albirostris) - 92% confidence
- **Monitor Lizard** (Varanus salvator) - 89% confidence

### Nature Reserves
- **Sungei Buloh Wetland Reserve**: Otters, Monitor Lizards, Waders
- **MacRitchie Reservoir**: Macaques, Hornbills, Butterflies
- **Pasir Ris Mangroves**: Otters, Crabs, Fish
- **Bukit Timah Nature Reserve**: Macaques, Bats, Birds

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Framework**: Bootstrap 5.3
- **Storage**: LocalStorage for session management
- **Architecture**: Single Page Application (SPA)
