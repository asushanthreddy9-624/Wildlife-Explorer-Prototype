# Wildlife Explorer Prototype

A web application for exploring wildlife species and their habitats.

## Features

- Browse wildlife species by category
- View detailed information about each species
- Search and filter functionality
- Interactive habitat maps
- Conservation status tracking

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Wildlife-Explorer-Prototype
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

This will open the application in your default browser at `http://localhost:3000`.

### Building

To build the application for production:
```bash
npm run build
```

### Testing

Run tests:
```bash
npm test
```

### Linting

Check code quality:
```bash
npm run lint
```
